Detailed Description

A struct type represents aggregate types that consist of a list of fields.

The following attributes are private to this type kind:

  • member: All entities belonging to this class. This are the fields that can have any of the following types: type_class, type_struct, type_union, type_array, type_pointer, type_primitive.
