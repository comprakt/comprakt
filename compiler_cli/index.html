[−][src]Crate compiler_cli

A command-line interface for the compiler_lib. If you are using this crate programatically, you probably want to call it as follows:

use compiler_cli::*;

let cmd = CliCommand::from_args();

if let Err(msg) = run_compiler(&cmd) {
    exit_with_error(&msg);
}
